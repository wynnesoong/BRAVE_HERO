# ⚔️ BRAVE HERO — 勇者傳說

## 🎮 遊玩方式

直接用瀏覽器開啟 `index.html` 即可遊玩，不需要伺服器或任何安裝。

```bash
# 方法 1：直接開啟
open index.html

# 方法 2：使用本地伺服器（可選）
npx serve .
```

## 🕹️ 操控

| 按鍵 | 動作 |
|:----:|:----:|
| `W` | 跳躍 |
| `A` | 向左移動 |
| `D` | 向右移動 |
| `S` | 開啟選單 / 商店 |
| `滑鼠左鍵` | 攻擊（揮拳 / 揮劍）|
| `R` | 重新開始（死亡後）|

## 📖 遊戲簡介

你扮演一位冒險家，在廣闊的世界中向前探索。途中會遭遇各種怪物，擊敗牠們可獲得戰利品與經驗值。每經過一段距離會遇到村莊，可以在商店買賣物資、補給回血，然後繼續踏上冒險之路。

隨著擊殺數與等級提升，怪物會越來越強、種類越來越多，你必須善用武器升級與道具補給來應對不斷升級的挑戰。

## 📈 難度系統

怪物解鎖採用**雙重門檻**機制：

1. **擊殺數門檻** — Tier 1 永遠可用，Tier 2 需 10 殺，Tier 3 需 25 殺，Tier 4 需 50 殺，Tier 5 需 80 殺
2. **血量門檻** — 玩家最大生命值必須大於怪物攻擊力的 2 倍

**生成權重分配：**
- 50% — 當前最高可用 Tier 的怪物
- 30% — 次高 Tier 的怪物
- 20% — 從所有可用怪物中隨機

**動態調整：**
- 場上怪物數量隨擊殺數增加（起始 1-3 隻，最多 10 隻）
- 移動中生成間隔更短（40 幀 vs 靜止 60 幀）
- 遠離超過 2.5 個畫面的怪物會被回收並在附近重新生成

## 🎨 技術特色

| 項目 | 說明 |
|:----:|:----:|
| 渲染 | HTML5 Canvas 2D，程式化像素風格繪製 |
| 音效 | Web Audio API 合成音效與音樂 |
| 音樂 | 村莊：鄉村風旋律 / 戰鬥：激昂背景曲 |
| 效能 | 背景快取至 offscreen canvas、音訊節點池（上限 12）、固定時間步進遊戲迴圈 |
| 粒子系統 | 打擊火花、死亡煙霧、升級光效、火焰/閃電特效 |
| 相依性 | **零** — 單一 HTML 檔案，無需任何框架或外部資源 |

### 效能優化

- 靜態背景（天空、星星、月亮）預渲染至 offscreen canvas
- 音訊並行節點上限 12 個，防止 GC 壓力
- 粒子/浮動文字/怪物陣列使用原地壓縮（write-index pattern），避免每幀產生新陣列
- 固定時間步進（16.67ms/幀），配合 `requestAnimationFrame`，防止幀率波動影響物理模擬
